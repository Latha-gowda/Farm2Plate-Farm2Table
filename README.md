🌾 Farm2Table
AI-Powered Farmer to Buyer Direct Marketplace
Farm2Table is a web platform that connects farmers directly with nearby buyers without middlemen. The system enables farmers to create communities, list crops, manage inventory, and sell products directly to buyers within a 10KM radius. Buyers can discover nearby farm products, place orders, and view AI-based insights about crop demand and farm activity.
The goal of Farm2Table is to increase farmer income, reduce middlemen dependency, and promote transparent farm-to-consumer trade.
🚀 Features
👨‍🌾 Farmer Features
Community System
Farmers can:
Join nearby farming communities
Create a new community if none exists nearby
Collaborate with other farmers in the same area
The community system uses GPS coordinates to group farmers geographically.
Crop Management
Farmers can:
Add crops/products
Set price per kg
Update quantity
Upload crop images
Manage their farm inventory
Community Inventory
All farmers in a community can view:
Available crops
Prices
Quantity available
This helps farmers avoid oversupply and coordinate selling.
Farmer Community Chat
Farmers can communicate with other farmers in the same community to:
Share farming tips
Discuss crop demand
Coordinate harvest and selling
AI Farming Insights Dashboard
Farmers get visual insights generated from system activity, including:
Crop demand trends
Price trends
Community activity
Monthly earnings
Profit summary
Low stock alerts
Graphs are displayed using Chart.js to make them easy to understand.
🛒 Buyer Features
Nearby Product Discovery
Buyers can:
Discover farm products within 10KM radius
View available crops
Check prices
Place orders directly with farmers
Order System
Buyers can:
Place orders
Track order status
View available stock
Farmers can:
Accept orders
Manage inventory after sales
AI Trust Check (Buyer Safety Feature)
Buyers can view an AI Trust Report that shows transparency indicators such as:
Crop image verification
Farmer transparency data
Harvest and storage information
Product trust score
This improves buyer confidence in the produce.
The trust score is currently calculated using rule-based logic. Future versions will integrate machine learning models for crop classification and freshness detection.
🌐 Multilingual Support
The system supports multiple languages:
English
Kannada
This makes the platform accessible for local farmers who may not be comfortable with English.
