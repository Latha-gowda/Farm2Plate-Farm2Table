 🌾 Farm2Table
### AI-Powered Farmer to Buyer Direct Marketplace

Farm2Table is a web application that connects *farmers directly with nearby buyers without middlemen*.  
The platform allows farmers to list crops, manage inventory, and sell products to buyers within a *10KM radius*.

This system improves *farmer income, transparency, and accessibility* in local agricultural markets.

---

## 🚀 Features

### 👨‍🌾 Farmer Features
- Join or create *farmer communities*
- Add crops with *price, quantity, and images*
- Manage *community inventory*
- View *AI farming insights dashboard*
- Chat with farmers in the same community
- Track and manage *buyer orders*

### 🛒 Buyer Features
- Discover farm products within *10KM radius*
- View crop details and prices
- Place orders directly with farmers
- View *AI Trust Check report* for transparency

### 🤖 AI Features
- *AI Farming Insights Dashboard*
  - Crop demand trends
  - Price trends
  - Community activity
  - Profit summary

- *AI Trust Check*
  - Crop image verification
  - Transparency indicators
  - Product trust score
